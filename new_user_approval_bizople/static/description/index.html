New User Sign-Up Approval & Verification

Go to Settings > General Settings > Users

In Customer Account - select option Free sign-up (B2C).

It allows your new users to get a sign-up option.

Sign-Up option avails to user.

User has to fill-in Sign Up Form.

On Sign-up, user gets notification email.

Administrator gets the user's details.

It tracks user's IP and Location for further reference.

Approve or Reject the new user on the basis of information.

On Approval or Rejection User will get notified.

Get notified about new user sign-up with setting email id in field "To whom(at admin side) you want to get notified at new Sign-Up?"

Set "Email Link Verification for New User Sign Up ?" option to verify user's email-id by sending sign-in link in mail and then only user will be able to sign-in for the first time.

A user without log-in can't see your product prices.

Once a user logs-in she can view the prices.
