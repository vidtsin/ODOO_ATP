Dynamic product label printing

This module will help you generate dynamic labels based on your own configuration, with different barcode, internal reference and custom logo

Contact us for getting trial or demo.

This module will provide with below features

  • Generate small labels
  • Configure small labels with attributes
  • Configure with number of quantities configured and generate labels
  • Configure labels based on Width and Length
  • Configure labels based on Margin required
  • Configure logo with Top or Bottom position with custom heigth and width
  • Configure barcode width and length
  • Configure barcode type by EAN13, EAN8 & Code128
  • Barcode can be generate based on Barcode or Internal Reference
  • Barcode can human readable configured based on products

To print labels you can go to Sales, Product Small menu and select Print small label as shown in screenshot and window will open

This menu will help you configure the labels As per screenshot numbering below explaination

  • Template : Set the template name or you can which is default
  • Update Existing template : Once its configured you can save those fields for future printing labels
  • Pricelist : Pricelist you can set to fetch the prices
  • Currency position : You can set the currency position Before or After the price
  • Design using : XML design which is been pre-configured which you can directly use for printing and another option is Field Selection which you can configured as per your requirements.
  • You can select the products for which you want to generate the labels, you can set the quantity based on that labels will be printed, there is attribute field selection which will print based attributes configured and selected.

In the screenshot Label which is highlighted in Red in that you can configure Label width, height, Margin (top, left, bottom and right), you can set custom logo of your company with position Top and Bottom with logo Height and Width.

In the screenshot Barcode tab is highlighted with Red will help you configure barcode width, Height

  • There are 3 different type of Barcode Type :
  • EAN13
  • code128
  • EAN8

If you want to see the barcode numbers you can select HumanReadable in the next screenshot you can see with EAN8 format, Barcode field you can select based on Internal Reference and Barcode field which both can set from products.

The highlighted fields in Red, Design Using : Fields selection will show Fields tab and configure which fields you want to print in labels.
